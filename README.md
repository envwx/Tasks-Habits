# Featuring in the project

* creating tasks
* adding habits
* users will be able to delete tasks/habits
* users will be able to edit both the tasks and habits
* allowing user to login and only show their tasks/habits
* while adding and creating habits & tasks users will be able to view them from the HomePage
* full separate pages for habits/tasks to show the full lists of each
* users will be able to complete their tasks/habits and it will be displayed
